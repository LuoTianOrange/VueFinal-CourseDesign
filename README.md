# 画作在线展览平台
## 项目结构
页面:IndexPage(主页),PhotoPage(图片详细界面),AuthorPage(作者详细页面)  
项目组件:CommentItem(评论区),DropDownTagTmp(首页下拉列表),ImgItem(画作展示盒),ImgTmp(画作悬停显示),MainFooter(底部栏),MainHeader(头部导航栏)
